# Влияние нормализации на прогнозирование временных рядов

## Описание проекта
Исследование влияния различных методов масштабирования на качество прогнозирования временных рядов для моделей разных классов

## Методы нормализации
- None (без нормализации)
- StandardScaler
- RobustScaler
- QuantileTransformer

## Ключевые выводы
1. Нейросетевые модели (PatchTST) критически зависят от нормализации входных данных
2. Деревья решений (CatBoost) устойчивы к масштабированию
3. Без нормализации нейросети показывают неприемлемо высокую ошибку (179%)
4. RobustScaler показал наилучший результат для PatchTST (1.66%)
